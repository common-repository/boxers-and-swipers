Integrates Colorbox, Slimbox, Nivo Lightbox and Photoswipe into WordPress.

== Description ==

= Integrates Colorbox, Slimbox, Nivo Lightbox and Photoswipe into WordPress. =
* WordPress 6.4 or later, it works if "LINK TO" is set to "Media FIle" in the block editor's gallery. It does not work when embedding media by itself.
* This plugin will only work if you specify the media in the <a\> tag.
* Can switch the effect to each user agent.
* Add effect to the WordPress gallery.

== Installation ==

1. Upload `boxers-and-swipers` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
